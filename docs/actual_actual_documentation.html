API documentation

The API can:

  • Request an article to be returned.
  • Add an artical.

GET article

This command returns an article when given the ID.
The format is 'API GET article/<ID> with the ID number of the article you want.
This returns the atricle as a .json file.

POST article

This command adds an article when given the title, sub heading and content
The request must be in the format
{
"title": <title>,
"sub_heading": <subheading>,
"content": <title>
}
