The actual docs for the API

/API

Notes:

You will be timed out for FAIL_TIMEOUT seconds (60s) after either:
  • MAX_FAIL (10) failed requests (ie., malformed data), or
  • MAX_WARN (3) violations of MIN_SEP (requests must be MIN_SEP seconds (.1s) apart).

GET /

Redirects to /docs

GET /API/article/<id>

Gets article #id, in form:

{
"id": ...,
"title": ...,
"sub_heading": ...,
"content": ...,
"date_published": ...
}

POST /API/article

Post an article; the following data is required:

{
"title": ...,
"sub_heading": ...,
"content": ...,
}
